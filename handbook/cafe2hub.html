Project site

https://github.com/zoom-quiet/wiki

这种,也就多一步:

  • 建立分支 git co -b hg_pages
  • 复制为 output 目录
  • 定制 fab 事务 ~ wiki/fabfile.py at master · zoom-quiet/wiki
    • 自动化这么几件事儿:
      • 用 markdoc 工具编译
      • 将 CNAME 复制到 output
      • 切换到 output 中将新编译结果用 git 命令推送到 github
  • 思路上参考: jekyll-to-pelican | #是也乎# | ZoomQuiet.io
